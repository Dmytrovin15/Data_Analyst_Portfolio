### 3. Аналіз ефективності рекламних кампаній (Facebook Ads)
* **Технології** Python (Pandas, Matplotlib, Seaborn).
* **Аналітичні завдання**
* Робота з часовими рядками: групування даних по днях та аналізвитрат і ROMI (Return on Marketing Investment).
* Статистичний аналіз: використання 'box plot' для визначення розкиду ROMI та гістограм для розподілу значень.
* Візуалізація: побудова теплових карт (Heatmap) для аналізу кореляції між показниками та використання лінійної регресії ('lmplot') для оцінки звязку між витратами та отриманою цінністю.
* **Ідея** Оптимізація маркетингових стратегій на основі глибокого аналізу даних та виявлення факторів, що найбільше впливають на прибутковість компаній.
